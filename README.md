# Calculator-App

The Arithmetic Calculator is a simple and interactive web-based tool that performs basic mathematical operations such as Addition, Subtraction, Multiplication, and Division. It is designed using HTML, CSS, and JavaScript, making it lightweight, responsive, and beginner-friendly.

--> Features

➕ Addition
➖ Subtraction
✖️ Multiplication
➗ Division

--> Real-time calculations

 Clear button to reset inputs

 Fully responsive and clean UI

 -->Technologies Used

HTML – For structuring the calculator layout

CSS – For styling and responsive design

JavaScript – For handling calculation logic
